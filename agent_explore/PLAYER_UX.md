# 玩家体验格式 · 留风镇

> **开放世界**的游玩实况：不是选项游戏，没有菜单。  
> 观察者在看「我」在世界里**自己琢磨、自己行动**。

---

## 我们在玩什么

不是 Galgame，不是二选一。

世界开着。我随时能回家、能开口、能站着不动。  
文字要写出**那一刻我心里过一遍什么**，然后**我实际做了什么**。

```
■ 场景 → 【现场】→ 【内心】（可长）→ → 行动与反馈 → 【变化】
```

---

## 场景行

```
■ 留风镇｜第1日｜夜｜风铃巷
```

---

## 现场

摄像机直述。少比喻，不诗化断句。

```
【现场】风铃同时停了。巷子安静。钟沉声说：听见了就听见了，别装。
```

---

## 内心（开放世界核心）

**大事、岔路口、想走又没走** — 用【内心】写我怎么权衡。

- 可以两三句，像真人脑子转
- 写「我想……但又……」「本来要……最后还是……」
- **不要**落成菜单：`（选择）· A · B`
- **不要**助手腔：「你可以……」

```
【内心】
我想转身回家。腿已经往那边偏了。
可钟沉声看着我。刚才那十秒太长，装没听见，以后巷子里更别扭。
开口又傻——就一个字。但我还是不想装。

→ 我说：「听见了。」
```

内心是**开放世界的操作过程**，不是系统弹窗。

---

## 行动与反馈

```
→ 我说：「听见了。」
→ 钟沉声点了一下头。别的风铃陆续响起来。
```

---

## 在场 / 变化

```
【在场】程晚禾、钟沉声、几个邻人
【变化】止铃十秒发生了。钟沉声记住你开口。
【之后】林归桐可能会问你脸为什么这么白
```

---

## 若人类亲自玩

观察者变成参与者：你随时插话「回家」「喊钟沉声」「站着别动」。  
Agent 不弹菜单，只续写世界反馈。

Agent 代玩时：用【内心】展示权衡，用 → 展示实际动作（可犹豫、可半道改主意）。

**每回合末尾**附 `📋 进度` 行（见 [SESSION.md](SESSION.md)），供用户续玩。

---

## 禁止

| 不要 | 要 |
|------|-----|
| （选择）· 选项列表 | 【内心】权衡 → 行动 |
| 小说比喻、诗化断句 | 直述 + 口语内心 |
| 助手「你可以」 | 我自个儿琢磨 |
