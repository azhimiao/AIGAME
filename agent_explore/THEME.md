# 主题 · 留风镇

> 设计者 / Agent 底层读。  
> **不对玩家直说下面加粗句。** 让玩家从息风季里自己走到。

---

## 最终主题（用户定稿）

> **我们这一代，最终要走出被他人评价的循环，自己寻找人生的意义。**

---

## 故事在讲什么

留风镇这个息风季，表面是风铃、榜单、拆铃、朋友要走。

底下只有一件事：

**你从出生就被风纹、风途、番付、祝灯、眼神定义。**  
谁值得投资、谁该远行、谁算落后、谁家进不去门——都是**别人替你写好的人生脚本**。

整个游戏不是问「留还是走哪个对」，而是问：

**你还要不要继续活在这套评价里？**  
**若不要，意义从哪来——不是名次给的，是你自己找出来的。**

---

## 他人评价的循环（世界观机制）

| 表现 | 在循环里 |
|------|----------|
| 风纹榜 | 红笔把你的价值贴给全镇看 |
| 风途适格 | 「上风」才配被祝灯照到 |
| 废止纹铃 | 你的日常被定为「滞后」 |
| 沈岫风空壳归来 | 按他们规则走出去，也可能走丢自己 |
| 祝灯不照我家檐 | 价值由**别人的光**决定 |
| 监察番记册 | 反抗也被收进档案 |

循环的两头都是陷阱：

- **按评价活**：末行自卑，适格狂喜，废止认命  
- **按评价反着活**：为证明「留也对」而留——仍是**他们的尺**

走出循环 ≠ 必须留镇 ≠ 必须出走。  
= **你的行动、你的注意、你的选择，开始由自己负责。**

---

## 意义从哪找（故事里已埋的落点，不说教）

不给标准答案。只给**可能找到意义的时刻**：

- 程晚禾归灯摇铃、林归桐留温食——**与名次无关的「有人要你活着」**
- 站温汐音旁边——**不是赢榜单，是选择谁值得并肩**
- 万铃寂夜敲铃——**在全网沉默时，自己决定「还要不要响」**
- 日记写下来——**意义先被你自己承认**
- 叫住知遥——**不替他选风途，只说不必一个人扛脚本**
- 挡拆铃——**不是因为官方说铃落后，而是你舍不得这条巷**

意义可以很小：一顿饭、一声铃、一行字。  
关键是：**这是你找的，不是榜给的。**

---

## 与五个结局

| 结局 | 与主题 |
|------|--------|
| **守夏者** | 在镇上找到自己愿意守的东西；不是因为末行认了命 |
| **叫住的人** | 打破「谁该一个人走」的脚本；意义在人与人之间 |
| **敲铃的人** | 在集体沉默里自己定义行动；被记名也接受 |
| **心已往别处** | 若跟知遥走是为徽章——**仍在循环**；若想清楚再走——**另一种自选**（叙事留白） |
| **晃过去的夏天** | 没走出循环；被评价推着，也没为自己找 |

Agent 写结局时：点出**守住了什么自己认的意义**，或**仍困在评价里**。

---

## 角色嘴里不要说的话

- 人生意义、打破循环、不要在乎别人、一代人使命  
- 知足、平等、反抗体制  

用事件让人感到，不要用演讲让人知道。

---

## 代玩 / 实况自检

1. 这一拍我是在**被别人定义**，还是在**自己琢磨**？  
2. 我的行动是为了**番付/眼神/祝灯**，还是**我自己认的一件事**？  
3. 玩家能否感到：留或走都是表面，**自主才是底**？
