# 留风镇 · 游戏介绍与 Agent 游玩说明书

> 给人看：**这是什么游戏**。  
> 给 Agent 看：**怎么代玩 / 怎么接玩家**。  
> 给人 + Agent：**怎么一起跑起来**。

---

# 一、游戏介绍（给人看）

## 这是什么

**留风镇**是一款**开放世界文字生活冒险**。

你不是来刷任务、不是来打副本。你是风途世界边缘一座小镇上的年轻人，在**息风季将尽**的十四天里，过日子、遇事件、失去一些人一些事、在几个瞬间自己拿主意。

**你是观察者**：看「我」在镇上生活。也可以随时插话，像开放世界一样说「去屋顶」「别开口」「跟温汐音站着」。

## 世界（一句话）

外面是**风途世界**——崇拜风不停、人用风纹排名。  
**留风镇**是风会停的地方，因此被轻贱，却有铜铃、息溪、食灯、归灯。

## 你要体验什么

- **异世界日常**：铜脊瓦、铃网、息溪、食灯铺、归灯三响  
- **阶级与评价**：风纹番付、祝灯不照某檐、纹铃废止审会  
- **震撼时刻**：全镇铃停十秒、万铃寂夜、同期空壳归来、陆知遥要走  
- **失去**：朋友、匠人、食灯铺、林归桐的笑容  
- **小反抗**：站温汐音旁边、敲第七枚、挡拆铃——可不赢  
- **底层主题**（不直说）：走出被他人评价的循环，自己找意义  

## 不是什么

- 不是乡土暑假、不是红烧收摊  
- 不是选项 Galgame（没有 A/B 菜单）  
- 不是小说（不要大段比喻散文）  
- 不是远征打怪（弱旅行感，故事在镇上）

## 五种结局（概略）

守夏者 / 叫住的人 / 敲铃的人 / 心已往别处 / 晃过去的夏天  
——取决于你怎么过这十四天，不是选对一项。

# 二、Agent 游玩说明书

## Agent 是谁

Agent **扮演**留风镇那个「我」：第一人称，在世界里行动。  
Agent **同时**把过程写成**游玩实况**给人类观察者看。

Agent **不是**导游、解说员、菜单系统。

## 启动前：Agent 要加载什么

1. **技能**：`agent_explore/SKILL.md`（Cursor 技能名 `live_explore`）
2. **会话协议**：`SESSION.md` — 日切、📋存档、结局运算、续玩
3. **首次必读**：`INTRO.md` → 用「我」念完，进入第1日午后
3. **内部参考**（不输出给用户）：
   - `world/timeline.md` — 第几日、什么阶段
   - `world/events.md` — 事件、结局条件、地点
   - `world/characters.md` — 人物
   - `PLOT_ANCHORS.md` — 每天必亮什么剧情
   - `WORLD_AESTHETIC.md` — 异世界用词，禁乡土
   - `DAILY_LIFE.md` — 每天至少一次归灯/食灯
   - `CLASS_LOSS_RESIST.md` — 阶级、失去、反抗
   - `THEME.md` — 主题（不对玩家直说）
   - `ATMOSPHERE.md` — 环境素材

## 每一回合怎么写（输出模板）

```
■ 留风镇｜第X日｜时段｜地点

【现场】……（摄像机，短句直述）
【在场】……（有谁）
【内心】……（开放世界：犹豫、想走又没走，可两三句）
→ ……（我做了什么）
→ ……（世界立刻回应）
【变化】……（大事后：世界变了什么）
【之后】……（可选：后面可能发生什么）
📔 ……（可选：日记一行）
📋 进度：第X日·时段·地点｜反抗N 躲开N 日记N｜……（每回合必附，见 SESSION.md）
```

### 日切与存档

- 当日夜戏结束：`—— 第X日 尽 ——`
- 用户截图/粘贴 `📋 进度` 行即可跨会话续玩（见 SESSION.md）

### 必须遵守

| 要 | 不要 |
|----|------|
| 第一人称「我」 | 第二人称「你」指玩家 |
| 【内心】权衡后行动 | `（选择）` A/B 菜单 |
| 异世界词：息溪、食灯、归灯、适格、审会 | 红烧、收摊、表哥、公听会、录取通知书 |
| 实况感 | 小说比喻、诗化断句 |
| 每天归灯/食灯底噪 | 只有大事件没有生活 |
| 爆点写重（见 PLOT_ANCHORS） | 温吞带过止铃、风纹榜 |

### 内心块 = 开放世界

不是弹选项。写「我想……但又……」「本来要……最后……」然后 `→` 写实际做的。

范例见 `samples/README.md`。代玩全剧时逐日对照锚点，第8日写重，第14日强制收束。

## 十四日剧情节奏（Agent 内部日程）

| 幕 | 日 | 必写大事 |
|----|-----|----------|
| 异响 | 1–2 | 第七枚不对；**第1夜全镇铃停十秒** |
| 裂缝 | 3–5 | **风纹榜**；沈岫风空壳归来；陆知遥适格 |
| 止铃 | 6–9 | **风途审会**；钟沉声倒；**第8夜万铃寂**；陆知遥走 |
| 落夏 | 10–14 | 拆队；食灯铺封；钟沉声死；结局 |

★ 全剧高潮：**第8夜万铃寂**（范例 `samples/day08_climax.md`）

## 结局何时触发

读 `SESSION.md` 的结局运算。  
第10日后可触发，第14日强制收束。  
用 `ENDING.md` 的文学化文本收尾，不说结局 ID。

## Agent 代玩流程（推荐）

```
1. 念 INTRO
2. 从 ■ 第1日｜午后｜家门口 开始
3. 每场景：现场 → 内心 → 行动 → 反馈
4. 一日至少：午后 / 黄昏 / 夜，4–6 节拍
5. 每日检查 PLOT_ANCHORS + DAILY_LIFE
6. 到爆点日写重，写【变化】
7. 回合末附 📋 进度；满足 SESSION 结局条件 → ENDING 收束
```

## Agent 禁止对用户输出

- 数据结构字段名 / 结局编号  
- 「你可以去……」菜单  
- 主题演讲、人生道理  
- 乡土词（见 WORLD_AESTHETIC）

# 三、人类怎么跟 Agent 玩

## 模式 A：纯观察（Agent 代玩）

你对 Agent 说：

> 用留风镇技能，从第1日开始代玩。按 PLAY_MANUAL + SESSION 输出实况，每回合给 📋 进度。  
> **samples 只是范例，不要照抄，每局重写对白。**

然后只看实况。想加速就说「跳到第3日」「写到万铃寂」。

**续玩**：粘贴上次 `📋 进度` 行 +「从下一节拍继续」。

## 模式 B：开放世界介入（推荐）

Agent 写到一半，你随时插话，例如：

- 「今天别去风铃巷，一直待在屋顶」
- 「风纹榜那天站温汐音旁边」
- 「万铃寂夜不要敲，转身回家」

Agent **不弹菜单**，根据你的话续写【现场】和【变化】。

## 模式 C：你当「我」

你对 Agent 说：

> 你只管世界和 NPC，我来扮演「我」。

你只发行动/对白，Agent 回世界反馈。

## 在 Cursor 里怎么启动（复制即用）

```
请加载 agent_explore/SKILL.md（live_explore），阅读 PLAY_MANUAL.md 和 SESSION.md。
从 INTRO 开始，用游玩实况格式代玩留风镇第1日。
每回合末尾附 📋 进度行。

samples/ 仅是范例：对照格式和锚点力度，不是剧本。
不要照抄范例对白与场面；每局根据 📋 进度和我的插话重写。

不要选项菜单，不要小说腔，不要乡土词。
```

### samples 是什么（给人看）

| 是 | 不是 |
|----|------|
| 给 Agent 看的**格式样本**、锚点要写多狠的**标尺** | 玩家要念的固定剧本 |
| 帮你判断「输出像不像游玩实况」 | 每局都会一模一样的台词 |

你每次开新局，实况内容应不同；只有 ■【现场】【内心】→ 和 📋 进度规则应稳定。
