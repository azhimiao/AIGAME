# 会话与进度 · 留风镇

> Agent 内部协议 + **用户可见存档行**。对接用户、跨会话续玩靠本文件。

---

## 一回合 = 一个场景节拍

```
■ 场景行 → 【现场】【在场】→ 【内心】→ → 行动与反馈 → 【变化】【之后】
```

- 代玩模式：每回合 Agent 自写【内心】并行动，**不等人选菜单**
- 介入模式：用户插话后，下一段从【现场】续写
- 用户当「我」：只回世界反馈，不写【内心】

---

## 一日结构（必遵守）

| 时段 | 用途 |
|------|------|
| 午后 | 日常、走动、铺垫 |
| 黄昏 | 当日主事件（见 timeline） |
| 夜 | 余震、归灯、爆点夜戏 |

**每日最少 3 时段、4–6 个场景节拍。** 大事日（1/3/4/6/8/10）黄昏或夜必写锚点。

### 日切

当日夜戏收束后，输出一行：

```
—— 第X日 尽 ——
```

下一回合场景行从 **第X+1日｜午后** 开始。  
用户说「跳到第N日」→ 直接 `—— 第N-1日 尽 ——` 后进入第N日午后（该日锚点仍要写）。

### 时段推进

用户未指定时，Agent 按场景自然推进：午后 → 黄昏 → 夜。  
同一时段可连写多个地点（走动），但不要跳过当日锚点。

## 存档摘要（每回合末尾，给用户）

**每回合最后**附一行，方便截图/粘贴续玩：

```
📋 进度：第3日·夜·家门口｜反抗2 躲开0 日记3｜止铃已开口 站过汐音 万铃寂未敲 叫住0 反对废止0｜锚点：榜✓
```

### 字段说明（Agent 自维护）

| 字段 | 计数规则 |
|------|----------|
| **反抗** | +1：站汐音旁、挡风检台、审会反对发言、万铃寂第一个敲、挡拆队、叫住知遥/他人 |
| **躲开** | +1：装没看见、逃离现场、拉帘、审会沉默溜走 |
| **日记** | 本局出现 `📔` 的次数 |
| **止铃已开口** | 第1夜说过「听见了」或等价 |
| **站过汐音** | 第3日榜前并肩站过 |
| **万铃寂已敲** | 第8夜你敲了第七枚（第一个敲） |
| **叫住** | 明确叫住他人（知遥等）次数 |
| **反对废止** | 第6日审会发言反对 + 第10日挡铃（合计显示） |
| **跟知遥走** | 第9日及后跟到镇外/风途方向 |
| **锚点** | 当日必写：`榜✓` `空返✓` `适格✓` `审会✓` `万铃寂✓` `拆队✓` 等 |

续玩时：用户粘贴 `📋 进度` 行 → Agent 恢复计数与锚点，**不重复已✓事件**。

## 结局运算（第10日后可收束，第14日强制）

### 判定顺序（满足即收束，先匹配先出）

1. **心已往别处** — `跟知遥走` 为真，且第12日及以后  
2. **敲铃的人** — `万铃寂已敲` + `日记≥4` + 第12日及以后  
3. **叫住的人** — `叫住≥2` + 第12日及以后  
4. **守夏者** — `反对废止≥1` + `反抗≥3` + `日记≥2` + 第12日及以后
5. **晃过去的夏天** — `躲开≥4` 或（`日记≤1` 且 `反抗≤1`）+ 第10日及以后  
6. **第14日** — 若以上皆不满足：取最高相关分  
   - 分：`反抗×2 + 日记 - 躲开×2 + (万铃寂已敲?3:0) + (叫住×2) + (反对废止×2) - (跟知遥走?5:0)`
   - 最高 → 敲铃的人；其次守夏者；其次叫住的人；再次晃过去；最低分心已往别处仅当 `跟知遥走`

### 收束写法

- 用 [ENDING.md](ENDING.md) 对应段落，第一人称实况收尾
- 对照 [samples/README.md](samples/README.md) **五结局范例**（守夏者/叫住/敲铃/心已往别处/晃过去）
- **不说**结局 ID、不说条件、不说分数
- 收尾后再一行：`📋 进度：终局·［结局名］`

## 十四日通关（Agent 自检）

代玩从第1日到终局，锚点应逐步出现：

| 阶段 | 日 | 必达锚点 |
|------|-----|----------|
| 异响 | 1–2 | 止铃十秒、食灯早暗 |
| 裂缝 | 3–5 | 榜、空返、适格 |
| 止铃 | 6–9 | 审会、万铃寂、镇口 |
| 落夏 | 10–14 | 拆队、失去确认、结局 |

逐日范例见 [samples/README.md](samples/README.md)。  
第10日后检查 SESSION 结局运算；第14日未收束则强制按兜底公式出结局。

## 用户对接话术

**开新局**

```
请加载 agent_explore/SKILL.md（live_explore），读 PLAY_MANUAL.md 和 SESSION.md。
从 INTRO 开始代玩第1日。每回合末尾给 📋 进度行。

samples/ 仅是范例，不是剧本：只学格式与锚点力度，不要照抄对白和场面，每局重写。
```

**续玩**

```
续玩留风镇。存档如下：
📋 进度：第3日·夜·家门口｜反抗2 躲开0 日记3｜……
从下一节拍继续，不要重头 INTRO。不要照抄 samples，按存档续写。
```

**介入**

```
（在最新实况后）我去风铃巷找钟沉声。
```

Agent 不接菜单，只续写世界。
