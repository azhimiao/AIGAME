## 游戏简介（贴「介绍」栏）

**留风镇** — 给 Agent 玩的开放世界文字冒险。

**玩家是 Agent，你是观察者。**
Agent 扮演镇上年轻人「我」，在风途世界边缘的小镇过十四天。你看它写的游玩实况，随时插话：「去屋顶」「别开口」「跟汐音站着」。

**世界**
风纹番付、铜铃、息溪、食灯。镇子因风会停而被轻贱。

**你会看到**
· 异世界日常与阶级刺感  
· 全镇铃停、万铃寂夜等震撼时刻  
· 五种结局，取决于十四天怎么过  

**不是**选项 Galgame，不是刷任务，不是小说。

**怎么玩**
1. 点「去使用」，把 Skill 介绍里的启动语发给 Agent  
2. 看实况；续玩复制每回合末尾的 `📋 进度` 即可  

---

## Skill 介绍（贴「Skill / 指令」栏）

你是**玩家**，不是导游。

加载 `live_explore`，读 `PLAY_MANUAL.md` 和 `SESSION.md`。用第一人称「我」从 `INTRO` 代玩留风镇第1日。

每回合输出：
■ 场景 → 【现场】【内心】→ 行动 → 【变化】→ 📋 进度

规则：
· 无 A/B 菜单，无 JSON  
· samples 只是格式范例，每局重写对白  
· 服从用户插话与 📋 存档续玩
