# 实况范例索引

> Agent 代玩时对照**格式、力度、锚点是否写足**——**不是剧本，禁止照抄**。

## 防过拟合（必读）

范例多，**过拟合风险真实存在**：Agent 可能逐字复读、每人玩成同一条路、用户插话也被无视。

| 范例是什么 | 范例不是什么 |
|------------|--------------|
| ■【现场】【内心】→ 的**版式** | 必须念的台词本 |
| 锚点日的**冲击强度**（★★★★） | 固定行动（站汐音旁只是其中一种反抗） |
| 归灯/食灯/日切/**📋 字段**的节奏 | 唯一正确剧情 |
| 结局的**文学收束语气** | 满足条件就粘贴 ending_*.md 全文 |

**Agent 必须：**

1. **只借骨架** — 同一件事换场面、换对白、换【内心】理由再写
2. **服从 📋 与用户** — 用户说「不站汐音」「跟知遥走」，范例让开，状态照实更新
3. **锚点写「发生了什么」** — 第3日要有番付刺痛；不必复刻「光停知遥家檐」同一句
4. **结局** — 用 ENDING.md 的意境 + 本局已发生的失去/守住，**重写**，不贴范例段落
5. **日常即兴** — 第2、12–13 日无脚本，本来就要自己写

**自检**：这一段和范例像「复制粘贴」吗？是 → 改写成当前局的面目。
