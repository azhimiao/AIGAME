## 七处可去

| 地点 | 异世界质感 | 连接 |
|------|------------|------|
| **家门口** | 风铃檐、归灯铜片、风灶光 | 屋顶、息溪岸、风铃巷、屋后旧径、**镇口** |
| **屋顶** | 铜脊瓦、铃网、息塔微光 | 家门口、风铃巷 |
| **息溪岸** | 石凉、细风鸣 | 家门口、屋后旧径 |
| **风铃巷** | 钟沉声修铃、食灯铺、铃网最密 | 家门口、屋顶 |
| **屋后旧径** | 旧风道、草深 | 家门口、息溪岸、**镇口** |
| **公校门前** | **风纹柱**、榜单张贴处 | 从家门口经石阶到达（不可从屋顶直跳） |
| **镇口** | 驿线起点、陆知遥告别处 | 家门口、屋后旧径 |

## 走动规则

- 相连地点可在一个时段内连续场景（写 `屋顶→石阶`）
- 公校门前、镇口需从家门口或旧径走过去，至少一个过渡节拍
- 开放世界：用户说去哪就去哪，但**当日锚点地点不能整段跳过**（可迟到、可改顺序，事件仍要发生）
