# 异世界观感 · 留风镇

> 留风镇是**风途世界边缘的聚落**。  
> 语感：**日漫异世界** + **完整中文姓名**（姓+名，对白用日常称呼）。

## 日漫语感

- **克制**：震惊用沉默，不大吵大闹
- **世界观内词**：适格、番付、审会、祝灯、滞风
- **不用**亲戚称谓、公听会、录取通知书

### 台词范例

```
监察番：「滞风者，别占风途名额。」
```

```
岫风站在檐下。风途徽章裂了。我叫他名字，他像没听过。
```

```
适格祝灯从巷头游过。光停在知遥家檐上。没照到我家。
```
